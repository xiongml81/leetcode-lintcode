# 224. Basic Calculator



Given a string `s` representing a valid expression, implement a basic calculator to evaluate it, and return _the result of the evaluation_.

**Note:** You are **not** allowed to use any built-in function which evaluates strings as mathematical expressions, such as `eval()`.

&#x20;

**Example 1:**

```
Input: s = "1 + 1"
Output: 2
```

**Example 2:**

```
Input: s = " 2-1 + 2 "
Output: 3
```

**Example 3:**

```
Input: s = "(1+(4+5+2)-3)+(6+8)"
Output: 2ä»Ž
```



```java
class Solution {
    public int calculate(String s) {
        Stack<Integer> stack = new Stack<>();
        int i = 0;
        int number = 0;
        int sign = 1;
        int subsum = 0;
        
        while(i < s.length()){
            char c = s.charAt(i);
            if(Character.isDigit(c)){
                number = 10 * number + (c - '0');
            }
            else if(c == '+'){
                // number *= sign;
                subsum += number * sign;
                sign = 1;
                number = 0;
            }
            else if(c == '-'){
                // number *= sign;
                subsum += number * sign;
                sign = -1;
                number = 0;
            }
            else if(c == '('){
                stack.push(subsum);
                subsum = 0;
                number = 0;
                stack.push(sign);
                sign = 1;
            }
            else if(c == ')'){
                subsum += number * sign;
                number = 0;
                sign = 1;
                subsum *= stack.pop();
                subsum += stack.pop();
            }
            i++;
        }
        if(number != 0){
            subsum += number * sign;
        }
        return subsum;
    }
}
```
