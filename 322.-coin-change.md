# 322. Coin Change

DFS

```java
class Solution {
    int res = Integer.MAX_VALUE;
    public int coinChange(int[] coins, int amount) {
        dfs(coins, amount, 0);
        if(res != Integer.MAX_VALUE){
            return res;
        }
        else return -1;
    }
    
    public void dfs(int[] coins, int amount, int index){
        
        if(amount < 0) return;
        
        if(amount == 0) {
            if(res > index){
                res = index;
            }
            return;
        }
        
        for(int i = 0; i < coins.length; i++){
            dfs(coins, amount - coins[i], index + 1);
        }
    }
}
```
