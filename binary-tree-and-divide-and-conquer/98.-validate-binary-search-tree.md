# 98. Validate Binary Search Tree



**中序遍历**

```java
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
class Solution {
    public boolean isValidBST(TreeNode root) {
        Integer inorder = null;
        Stack<TreeNode> stack = new Stack<>();
        TreeNode curr = root;
        while(!stack.isEmpty() || curr != null){
            while(curr != null){
                stack.push(curr);
                curr = curr.left;
            }
            curr = stack.pop();
            if(inorder != null && curr.val <= inorder) return false;
            
            inorder = curr.val;
            curr = curr.right;
            
        }
        return true;
    }
}
```



**递归**

```java
class Solution {
    public boolean isValidBST(TreeNode root) {
        return dfs(root, null, null);
    }
    
    public boolean dfs(TreeNode root, Integer lower, Integer upper){
        if(root == null){
            return true;
        }
        
        if(lower != null && lower >= root.val) return false;
        if(upper != null && upper <= root.val) return false;
        
        return dfs(root.left, lower, root.val) && dfs(root.right, root.val, upper);
    }
}
```
